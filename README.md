# 🚀 Utility Billing Calculator (ASP.NET Core MVC Web App)

A modern **ASP.NET Core MVC Web Application** that calculates municipal water bills using tier-based pricing and automatic surcharge rules.

This project demonstrates clean MVC architecture, structured business logic separation, and production-style project organization.

---

## 🌟 Primary Application

> The main implementation is the **Web Application (MVC)** version.

It provides:

- Clean browser-based interface  
- Tier-based billing calculation  
- Automatic surcharge application  
- Input validation  
- Structured MVC architecture  

---

## 🧮 Billing Logic

The application calculates water bills using:

- **Tier 1:** 0–10 units @ R5 per unit  
- **Tier 2:** 11–30 units @ R8 per unit  
- **Tier 3:** 31+ units @ R12 per unit  
- **10% surcharge** applied when usage exceeds 50 units
